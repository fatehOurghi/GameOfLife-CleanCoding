# Game Of Life-CleanCoding
Implementation of game of life using design patterns
This is a Netbeans swing GUI project, that implements several design patterns like: 
### Lookup(Service Locator)
### Abstract Factory
### Observer
### Composite
### Delegator
### Mediator
### Memento
### Facade
